# Event Search System

This project is a web application built using Django REST API and HTML/JavaScript.

## Features
- Search events from CSV files
- Filter by start time and end time
- Displays results with event name, time, and file
- Shows search execution time

## Technologies Used
- Python
- Django
- Django REST Framework
- HTML
- JavaScript

## Dataset

The project includes an archive file:

data_files/events.tgz

This file contains the original dataset of event logs.  
For demonstration and easier testing, the application currently reads from CSV files:

- data_files/events1.csv
- data_files/events2.csv

These CSV files contain sample event records used to demonstrate the search functionality.
